This is a Dialog Popup. The site's content wrapper is given an `aria-hidden` attribute in order to hide the main content while the Dialog is open. The "Close" button is set by passing the element to the `setCloseButton` instance method.

The `useHiddenAttribute` option is set to `false` here to allow the CSS transition to be visible in both states.

Keyboard focus is "trapped" within the Dialog, which prevents elements outside the Dialog from gaining focus until the Dialog is closed.
